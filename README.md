StateLayout
=========

介绍
---

状态布局，暂时仅包括状态帧布局；
状态帧布局，通常用于网络请求的四种状态，普通、载入、错误、空白。支持Drawable或者View来展示，也可以混搭。

使用
---

**引用:**
```
dependencies {
    ...
    implementation 'io.github.alexmofer.widget:statelayout:1.0.0'
    ...
}
```
